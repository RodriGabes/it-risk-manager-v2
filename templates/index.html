ITRM v2

IT Risk Manager by RodriGabes

Esta interface é capaz de cadastrar, consultar, modificar e excluir entradas de um banco de dados que representa ativos de uma infraestrutura de TI, bem como o cadastro de responsáveis e vulnerabilidades associadas, auxiliando a gestão de riscos. Este projeto integra as atividades acadêmicas desenvolvidas duranto o curso de Bacharelado em Cibersegurança - UFU (I Turma).
